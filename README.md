# Hotel Reservation System Using JDBC and Servlets

## Description
A Java-based Hotel Reservation System developed using JDBC and Servlets. The application allows users to manage hotel reservations, store customer details, and interact with a database efficiently.

## Features
- Add New Reservation
- View Reservation Details
- Manage Customer Information
- Database Connectivity using JDBC
- User-Friendly Interface

## Technologies Used
- Java
- JDBC
- Servlets
- HTML
- MySQL / HSQLDB
- Apache Tomcat

## Requirements
- Java JDK 8 or above
- Apache Tomcat 10
- MySQL or HSQLDB Database
- JDBC Driver
- VS Code

## Project Structure
```
Hotel-Reservation-System/
├── InitDB.java
├── DBConnection.java
├── ListReservations.java
├── RS.java
├── index.html
├── hotel_db.properties
├── hotel_db.script
├── WEB-INF/

```

## How to Run
1. Clone the repository.
2. Configure the database.
3. Start Apache Tomcat Server.
4. Deploy the project.
5. Open the application in your browser.
